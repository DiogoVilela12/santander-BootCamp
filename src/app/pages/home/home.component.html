Santander Bootcamp 2023

Fullstack Java e Angular

O Santander Bootcamp 2023 é um programa de treinamento projetado para impulsionar as habilidades e o conhecimento de desenvolvedores. Neste bootcamp, os participantes terão a oportunidade de mergulhar profundamente no mundo do desenvolvimento Fullstack, explorando as tecnologias Java e Angular.

Contribuindo em um Projeto

Open Source no GitHub

Ao contribuir, os desenvolvedores têm a oportunidade de colaborar com uma comunidade, compartilhar suas habilidades e conhecimentos, e ajudar a melhorar o projeto por meio de correções de bugs, implementação de recursos e aprimoramentos.

Construindo uma Pokédex

com TypeScript e Angular

Neste projeto, o desenvolvedor ira aplicar as ferramentas TypeScript e Angular para construir uma aplicação que permite explorar informações sobre Pokémon, como suas características, tipos e imagens.

Criando um Blog

com Angular

Os desenvolvedores aprenderam a configurar o ambiente de desenvolvimento Angular, criar componentes para gerenciar postagens, implementar recursos como comentários e categorias, e integrar APIs ou serviços para armazenar e recuperar dados do blog.

Criando a PlayStation Store

com Angular

Neste projeto, os participantes são desafiados a recriar a interface da PlayStation Store, uma plataforma de comércio eletrônico de jogos e conteúdo relacionado à PlayStation.

Projetos desenvolvidos

Em Java

Esses projetos abrangem uma variedade de áreas, desde sistemas de gerenciamento de banco de dados, aplicativos web e muito mais. Projetos desenvolvidos em Java são frequentemente valorizados por sua portabilidade, desempenho e segurança.
